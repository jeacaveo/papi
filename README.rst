Papi
====

REST API for Prismata related data.

Install as library
--------------------

    **pip install papi**

Development environment
-----------------------

0. Virtual environment (look at virtualenvwrapper):

   Create:

    mkvirtualenv papi

   Activate:

    workon papi

1. Install dependencies:

    pip install -r requirements-dev.txt

2. Run tests:

    ./runtests.sh

3. Run linters:

    ./runlinters.sh
